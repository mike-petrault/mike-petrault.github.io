I work on enhancing the performance of codes used in cosmology. A first aspect of my thesis is the parallelisation of a code reducing bias on galaxy observations catalogs, and the challenges associated with the dependencies inherent to the MCMC processes used. The aim is to have a code that can efficiently handle million-galaxy catalogs that can then be used to contrain cosmological simulations. Another aspect is the implementation of subgrid physics modules in the hydrodynamics simulation code Dyablo, which can run on multiple architectures (CPU, GPU). In particular, the challenge resides in efficiently dealing with the non-locality of some of the physical phenomenon in a parallel setting.
Keywords: High Performance Computing, Shared-memory parallelism, GPU Computing, Astrophysics, Cosmology
